# Grocery List Management System

## Description
This is a simple Java console application to manage a grocery list using **arrays**. Users can add, display, search, update, delete, count, and sort items in their grocery list.  

## Features
- Add items to the list
- Display all items
- Search for an item
- Update an existing item
- Delete an item
- Count total items
- Sort items alphabetically

## Technologies Used
- Java SE
- Arrays for storing grocery items
- Scanner class for user input
